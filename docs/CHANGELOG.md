## 版本号规范

Fab Helper 脚本遵循 [语义化版本控制](https://semver.org/lang/zh-CN/) 规范：

- **主版本号**：当进行不兼容的 API 更改时增加
- **次版本号**：当添加向后兼容的新功能时增加
- **修订号**：当进行向后兼容的问题修复时增加

额外的标签：
- **alpha**：早期测试版本，功能不完整
- **beta**：功能完整但可能不稳定
- **rc**：候选发布版本，即将正式发布

例如：
- 1.0.0-alpha.1
- 1.0.0-beta.2
- 1.0.0-rc.1
- 1.0.0
